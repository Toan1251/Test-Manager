# Hệ thống quản lý lịch thi

## Enviroment

- Nodejs >= 16.0.0
- MySQL >= 8.0.0

## Set Up

### Server

Vào `cmd`, chạy lần lượt các lệnh

    cd ./backend
    npm install
    echo a > .env

Vào File .env vừa tạo, tạo các biến môi trường `PORT`, `DB_NAME`, `DB_HOST`, `DB_USER`, `DB_PASSWORD`, `DB_DIALECT`, `SESSION_SECRET`, `ENDPOINT`
